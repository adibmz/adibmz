## 📌 Featured Projects

### ⭐ CityCare — Civic reporting demo  

**What it does:** A demo civic-reporting app that lets residents submit urban incident reports, view them on a map or list, and lets city staff triage from an admin kanban dashboard.  
**Stack:** React 18, Vite, Tailwind CSS, Leaflet (React Leaflet)  
**Highlights:** Multi-step report creation with photo preview and map-based location picking; map & list views with search, filters, sorting, and pagination; admin dashboard with kanban workflow and seeded demo data.  

Repo: https://github.com/adibmz/city-care

### ⭐ Book Notes — Full-stack reading notes manager  

**What it does:** A full-stack web app to add, edit, review, and sort your book reading notes with support for rating, cover images, and rich metadata.  
**Stack:** Node.js, Express, EJS, PostgreSQL  
**Highlights:** Persistent storage in PostgreSQL, cover image fetch by ISBN, responsive UI, sort & filter by title/rating/date. Ideal for demonstrating real DB-backed CRUD and deployable server code.  

Repo: https://github.com/adibmz/book-notes

### ⭐ Simple EJS Blog Website — CRUD blog demo  

**What it does:** A simple blog built with Node.js, Express, and EJS — supports creating, updating, and deleting posts with a clean responsive UI.  
**Stack:** Node.js, Express, EJS  
**Highlights:** Straightforward CRUD flows, clear project structure, easy-to-run demo (good for showing server-side rendering and basic app structure).

Repo: https://github.com/adibmz/blogger-website
